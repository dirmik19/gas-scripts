# CLAUDE.md

このファイルは、Claude Code (claude.ai/code) がこのリポジトリで作業する際のガイドラインです。

## プロジェクト概要

- **プロジェクト名**: gas-scripts
- **内容**: Google Apps Script (GAS) 関連のスクリプトを管理するプロジェクト

## Git運用ルール

- **コードに変更を加えるたびに、必ずGitHubへプッシュすること。**
  - 変更内容ごとに適切な単位でコミットを作成する。
  - コミットメッセージは変更内容が分かるように簡潔に記述する。
  - コミット後は速やかに `git push` を実行し、リモート(GitHub)へ反映する。
  - 作業を中途半端な状態のままローカルにのみ残さない。
- コミット・プッシュ前に `git status` / `git diff` で変更内容を確認する。
- 意図しないファイル(認証情報、`.env` など)を誤ってコミットしないよう注意する。
- 破壊的な操作(`git reset --hard`、`git push --force` など)は明示的な指示がある場合のみ実行する。

## 開発上の注意事項

- Google Apps Scriptの認証情報やAPIキーなど機密情報はコードに直接書かず、Script PropertiesやPropertiesServiceを利用する。
- スクリプトの変更は、可能であれば実行して動作確認を行ってからコミットする。
